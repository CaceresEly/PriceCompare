# PriceCompare

A fullstack price tracking app with WhatsApp alerts.

## Features
- Product search
- Price comparison
- Price intelligence (best offer, average)
- Price alerts via WhatsApp
- Automated checks with cron jobs

## Tech Stack
- Next.js
- Prisma + PostgreSQL (Neon)
- Twilio (WhatsApp API)
- Vercel
